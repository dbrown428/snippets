
# Snippets

Repetitive code snippets from various projects.

## VSCode

### Setup

1. Place the contents of this directory in `~/Library/Application Support/Code/User/snippets`.
2. Familiarize yourself with the snippets for each language.
3. Each snippet has a **prefix** that is used for lookup.

### Insert

1. Start typing the **prefix** of the snippet you want to insert.
2. You should see a drop down list with matches.
3. Select the snippet you need, and press **return** or **tab** to insert the snippet into your document.
